# 🔄 VLESS Converter

Веб-конвертер конфигов Koala Clash и Happ JSON в VLESS-ссылки.

## 🌐 Демо

[Открыть конвертер](https://ВАШ_ЛОГИН.github.io/vless-converter/)

## ✨ Возможности

- ✅ Конвертация **Koala Clash** конфигов (YAML) → VLESS
- ✅ Конвертация **Happ JSON** конфигов → VLESS  
- ✅ Поддержка всех VLESS-параметров: TLS, Reality, WebSocket, TCP
- ✅ Массовая конвертация — все серверы сразу
- ✅ Копирование одной ссылки или всех сразу
- ✅ Скачивание всех ссылок в `.txt` файл
- 🔒 Работает полностью в браузере — данные никуда не отправляются

## 🚀 Использование

1. Откройте [демо-сайт](https://ВАШ_ЛОГИН.github.io/vless-converter/)
2. Выберите вкладку **Koala Clash** или **Happ JSON**
3. Вставьте конфиг в текстовое поле
4. Нажмите **"Конвертировать в VLESS"**
5. Копируйте нужные ссылки!

## 📱 Работает на

- 💻 Desktop (Windows, macOS, Linux)
- 📱 Mobile (iOS, Android)
- 🌐 Все современные браузеры
